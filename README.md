# Página Web Básica

Este repositorio contiene una página web sencilla creada como actividad académica.

## Características del proyecto

- Repositorio público con ramas:
  - main
  - desarrollo
  - release
- Más de 5 commits realizados
- Proyecto con HTML y CSS

## Estructura de la página

- **Header**: menú, título y subtítulo  
- **Body**: título, párrafo, dos imágenes con botones hacia otras páginas  
- **Footer**: dirección, correo, teléfono y enlace a Google Maps

## Tecnologías

- HTML 
- CSS
- Git & GitHub
